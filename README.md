# 100 Days of Python

Welcome!

This repository documents my journey through the 100 Days of Python course. I'm using it to stay consistent, track my progress, and build a strong foundation in Python through daily practice.

Inside this repository you'll find notebooks, coding exercises, mini projects, and personal notes from each day of the course. As I move forward, the projects become more hands on and cover topics ranging from Python fundamentals to automation, web development, APIs, data analysis, and more.

This isn't a collection of perfect code. It's a record of learning, experimenting, making mistakes, and improving one day at a time.

If you're on a similar journey, feel free to explore the code and learn along with me!
